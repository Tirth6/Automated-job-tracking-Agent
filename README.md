# Automated-job-tracking-Agent
An automated data extraction and tracking workflow built with n8n and OpenRouter to transform unstructured web URLs into clean tabular database sheets.

🤖 Automated Job Tracking Agent using n8n & AI
An end-to-end automated workflow built with n8n that completely eliminates manual data entry for job application tracking. The system automatically extracts unstructured data from web links, parses the contents using an LLM, and logs clean, structured records directly into a spreadsheet in real-time.

🚀 Key Features & Workflow Steps
Automated Data Scraping: Triggers on a set schedule to fetch raw, unstructured metadata from target web URLs.
Data Control Integration: Uses conditional data nodes to limit, filter, and throttle traffic safely.
AI-Powered Parsing: Integrates an AI Agent connected via OpenRouter to read messy text blocks.
Structured Output Parsing: Automatically extracts critical data fields like Job Title, Company Name, and Application Link.
Real-Time Sync: Automatically appends or updates data directly into a structured spreadsheet.

🛠️ Tech Stack & Tools Used
Workflow Automation: n8n
LLM Connection: OpenRouter 
APIData Storage: Google Sheets / ExcelCore 
Logic: Structured Output Parsers, Conditional Logic Nodes

Project Impact
60% Efficiency Gain: Successfully cut manual application tracking and entry times from 20 minutes down to under 8 minutes per cycle.
Data Consistency: Achieved 100% automated formatting without needing human data cleansing.
